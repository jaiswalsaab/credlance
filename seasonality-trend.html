Expected Format — Tally Monthly Export

  • Export via: Gateway → Display → Accounts Books → Sales/Purchase Register → F2 [set 24–36 month range] → Alt+N Monthly → Alt+E Excel
  • Column A: Month name (e.g. "April 2023", "Apr-23", "1-Apr-23")
  • Column B: Value (may include commas — automatically handled)
  • Header rows (company name, report title) are auto-skipped
  • Grand Total rows at the bottom are auto-excluded
  • Indian FY (April–March) supported natively · Values in INR
