Accepted Tally Exports

  • Cost Centre Break-up: Gateway → Display More Reports → Statements of Accounts → Cost Centres → Category Summary → Alt+E
  • Day Book with cost centre column: Gateway → Display → Day Book → F12 Show Cost Centres: Yes → Alt+E
  • Trial Balance with cost centre: Gateway → Display → Trial Balance → F12 Show Cost Centres: Yes → Alt+E
  • Expected columns: Cost Centre, Ledger/Group, Amount (or separate Income/Expense amounts)
  • Pre-requisite: Cost Centres enabled in Tally (F11 → Accounting Features → Maintain Cost Centres: Yes)
